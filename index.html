HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1 to h6 are the various header sizes, h1 should only be at the top
  • Meta data helps render the page correctly
  • HTML provides the framework of a webpage

CSS

  • The most common way to style a webpage is by using an external CSS style sheet.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • Use an Asterisk to apply a rule to all the elements visible on the page.
  • The box-shadow CSS property adds shadow effects around an element's frame.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls latest main branch
  • git push origin "item name": used to make changes to the github branch
  • git add -A: adds all changes from working branch to main upload

JavaScript

  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • JavaScript is case sensitive.
  • A semicolon at the end of a line indicates where a statement ends.
  • An operator is a mathematical symbol that produces a result based on two values or variables.
  • use the console.log method to output a message to the web console by adding an argument.
  • A function is a set of instructions that tells the computer how to perform a certain task.
